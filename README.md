# Stringed

A library for Stringed instruments. Recognizes chords and notes for any stringed instrument.

## Installation

Add this line to your application's Gemfile:

    gem 'stringed'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install stringed
